##  AngularJS

- 2 way data binding
```html
<input ng-model="yourName" 
placeholder="Enter a name here"
 type="text">
<h1>Hello {{yourName}}!</h1>
```

- controllers, service, view
- modules
  - ngResource
  - ngTranslate
  - ngRoute

note:
- most popular javascript framework
- powerful & easy to learn
